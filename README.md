# FirstProject
🏋️ Fitness ve Beslenme Asistanı (C)

Bu proje, C programlama dili kullanılarak geliştirilmiş konsol tabanlı bir fitness ve beslenme asistanıdır. Kullanıcıların antrenman ve beslenme süreçlerini planlamalarına yardımcı olmak amacıyla çeşitli hesaplamalar ve hazır programlar sunar.

🚀 Özellikler
🥗 Diyet Programı Hesaplama
Kilo alma, koruma ve verme seçenekleri
Günlük kalori ve makro (protein, karbonhidrat, yağ) hesaplama
Günlük su ihtiyacı hesaplama
📊 Vücut Değerleri Hesaplama
BMR (Bazal Metabolizma Hızı)
BMI (Vücut Kitle İndeksi)
BF (Yağ Oranı)
📈 Progressive Overload Analizi
Haftalık ağırlık artış yüzdesi hesaplama
Gelişim takibi
🏋️ 1RM ve Antrenman Hacmi
Maksimum tekrar (One Rep Max) hesaplama
Toplam antrenman hacmi hesaplama
📋 Hazır Antrenman Programları
Hipertrofi programları (Başlangıç / Orta / İleri)
Kuvvet programları (Başlangıç / Orta / İleri)
📚 Bilgi Paketi
Fitness terimlerinin açıklamaları (BMI, BMR, 1RM, vb.)
🛠️ Kullanılan Teknolojiler
C Programlama Dili
Standart C Kütüphaneleri (stdio.h, math.h, string.h)
Windows Console API (windows.h)
